#  Sarthak Maurya | Portfolio

My personal portfolio showcasing my projects, skills, and experience. Designed to be minimal, responsive, and blazing fast.

###  Get It Running

```bash
# 1. Clone the repository
git clone [https://github.com/sarthexe/portfolio.git](https://github.com/sarthexe/portfolio.git)

# 2. Navigate to the directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Ignite 
npm run dev
